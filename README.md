# LinkDetoxer 🔗

A privacy-focused web application that removes tracking parameters from URLs, giving you clean links to share.


[![Website](https://img.shields.io/website?url=https%3A%2F%2FLinkDetoxer.angelmakes.software&label=LinkDetoxer&style=for-the-badge&color=14b8a6)](https://LinkDetoxer.angelmakes.software)

## 🌟 Features

- **Universal Tracking Removal**: Removes tracking parameters from any URL
- **Privacy-First**: All processing happens client-side - no data sent to servers
- **Real-time Statistics**: Track usage stats with local storage
- **Modern Dark UI**: Teal dark mode design
- **Mobile Responsive**: Works perfectly on all devices
- **Fast & Lightweight**: Pure vanilla JavaScript, no frameworks
- **Keyboard Shortcuts**: Ctrl/Cmd+K to focus, Escape to clear
- **Copy to Clipboard**: One-click copying of clean URLs

## 🔧 Technical Implementation

LinkDetoxer implements the most comprehensive tracking detection system available:

### **Core Detection Methods**
1. **Exact Parameter Matching**: 2000+ hardcoded tracking parameters from every major platform
2. **Pattern-Based Detection**: Smart regex patterns for dynamic tracking parameters
3. **Fragment Cleaning**: Removes tracking data from URL fragments/hashes
4. **International Support**: Handles non-Latin characters and international domains

### **Parameter Categories**
- **600+ Social Media** tracking parameters (Facebook, TikTok, Instagram, etc.)
- **350+ E-commerce** tracking parameters (Amazon, Shopify, regional stores)
- **450+ Analytics & Marketing** tracking parameters (Google, Adobe, HubSpot, etc.)
- **250+ International** tracking parameters (Chinese, Japanese, Korean, Russian platforms)
- **300+ Mobile & App** tracking parameters (attribution networks, deep links)
- **250+ Specialized** tracking parameters (gaming, crypto, travel, dating)

### **Advanced Features**
- **Zero False Positives**: Carefully curated parameter list avoids removing functional parameters
- **Performance Optimized**: Efficient parameter matching for instant results
- **Privacy Focused**: All processing happens client-side, no data transmission
- **Future Proof**: Easy to extend with new tracking parameters as they emerge

## 📊 Statistics Tracking

LinkDetox tracks the following metrics locally:
- Total links detoxed
- Total site visitors
- Monthly visitors
- Total tracking parameters removed

All statistics are stored locally and never sent to external servers.
